# Airsoft-Bomb-Arduino
A countdown bomb with LCD Screen, keyboard and password to defuse. Inside the main folder there's everything you need to make it work. It has been developed on year 2021 for a friend who liked Airsoft, but since then I haven't compiled it yet. If you have a question or problem I will be glad to help.
